El día de hoy trabajamos en la parte del backend de nuestro proyecto de tienda 
de música, específicamente en la configuración de la base de datos MySQL.

Lo que hicimos:

1. CONFIGURACIÓN DE CONEXIÓN A BASE DE DATOS (db.js)
   - Creamos el archivo backend/src/config/db.js
   - Instalamos el paquete mysql2 para poder conectar con MySQL
   - Configuramos la conexión usando variables de entorno
   - La conexión incluye: host, usuario, contraseña y nombre de la base de datos
   - Agregamos manejo de errores para saber si la conexión falla

2. VARIABLES DE ENTORNO (.env)
   - Creamos el archivo backend/.env para guardar configuraciones sensibles
   - Definimos: PORT=3000 (puerto del servidor)
   - DB_HOST=localhost (servidor de base de datos)
   - DB_USER=root (usuario de MySQL)
   - DB_PASSWORD= (contraseña - vacío en este caso)
   - DB_NAME=tienda (nombre de la base de datos)

3. CONTROLADOR DE PRODUCTOS (productos.controller.js)
   - Creamos el archivo backend/src/controllers/productos.controller.js
   - Implementamos la función getProductos que consulta todos los productos
   - Usa db.query() para ejecutar SQL directamente
   - Retorna los resultados en formato JSON
   - Maneja errores con código 500 si la consulta falla

4. RUTAS DE PRODUCTOS (productos.route.js)
   - Creamos el archivo backend/src/routes/productos.route.js
   - Definimos la ruta GET /api/productos
   - Conectamos la ruta con el controlador correspondiente

5. ESQUEMA DEL MODELO (producto.model.js)
   - Creamos el archivo backend/src/models/producto.model.js
   - (Archivo preparado para futuras implementaciones del modelo)

6. APLICACIÓN PRINCIPAL (app.js y server.js)
   - Configuramos Express en backend/src/app.js
   - Agregamos CORS para permitir peticiones desde el frontend
   - Configuramos express.json() para recibir datos en JSON
   - Conectamos las rutas bajo el prefijo /api
   - Creamos el servidor en backend/src/server.js
